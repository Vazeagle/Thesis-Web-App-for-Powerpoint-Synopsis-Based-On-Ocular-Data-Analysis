This is a Web Application created by Stylianos Vazaios in the context of his thesis "Creation of a Web appplication for automatic presentation summary based on users ocular data" used to automatically create a synopsis of a powerpoint file based on ocular data analysis

Used Technologies:

In general python was used for almost everything, mostly the backend."

There are python scripts to connect with Tobii eyetracker and save the data on csv files based on the slide number

Also with python-pptx library for presentations and some "smart" functions we can get the precice location on a screen of the elements of the powerpoint like images textboxes groups, etc.

Python Flask was used to run the web server to show the html and css files and to connect to the created Database.

For the database XAMPP was used.
